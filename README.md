
# Airbnb Project #

## Tools and Techniques Used 
1. Front-end: HTML CSS & Bootstrap JavaScript
2. Database: Node.js Express.js
3. Backend: MongoDB 
4. Docker Deployment for MongoDB 

    `docker-compose up`

### Deployment :

1. Atlas for MongoDB 

2. Deploy project Render
